## SUPERSEDED

This change has been superseded by `remap-session-cycle-keys`.

- The old `Ctrl+b N` / `Ctrl+b P` prefix bindings were replaced by root-table `Ctrl+.` / `Ctrl+,`.
- The `aoe tmux switch-session --global` flag was removed; directional cycling is now always global.
- The current behavior lives in `openspec/changes/remap-session-cycle-keys/specs/root-key-session-cycle/spec.md`.
